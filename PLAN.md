# Achoo — 글로벌 꽃가루/알레르기 예보 앱

> 앱 이름: **Achoo**
> 목표: 구글 광고(AdMob) 수익 월 200유로
> 타겟: 글로벌 (위치 기반, 언어 영어)
> 플랫폼: 모바일 앱 (iOS + Android) + 선택적으로 웹

---

## 핵심 개념

"날씨 앱처럼 꽃가루 예보를 받고, 오늘 외출 전에 알레르기 대비를 할 수 있는 앱"

- 위치 기반 실시간 꽃가루 지수 확인
- 매일 아침 푸시 알림 ("오늘 서울 꽃가루 매우 높음 🤧")
- 알레르기 유형별 대처법 제공
- 디자인 최소화 — 심플한 숫자와 색상 코드로 표현

---

## 핵심 기능 (MVP)

### 1. 홈 화면
- 현재 위치의 꽃가루 지수 (낮음/보통/높음/매우높음)
- 꽃가루 종류별 분류 (나무, 잔디, 잡초)
- 7일 예보

### 2. 푸시 알림
- 매일 아침 설정 시간에 오늘 꽃가루 지수 알림
- 지수가 "높음" 이상일 때 긴급 알림 (선택)

### 3. 알레르기 관리 팁
- 꽃가루 높은 날: 외출 시간대, 마스크 착용, 환기 금지 등
- 실내 관리: 공기청정기, 창문 닫기 권고 시간

### 4. 광고
- 홈 화면 하단 배너 광고 (AdMob)
- 팁 페이지 로드 시 인터스티셜 광고

---

## 수익 목표 계산

| 지표 | 목표값 |
|------|--------|
| DAU | 1,000명 |
| 세션당 광고 노출 | 1.5회 |
| 인터스티셜 eCPM | $4 (글로벌 평균) |
| 월 예상 수익 | ~$180-220 (≈ 170-200유로) |

- 핵심: Tier 1 국가(미국, 영국, 독일) 사용자 비율 높을수록 수익 상승
- 꽃가루 시즌(봄/가을) 트래픽 2-3배 폭발 → 연평균으로 목표 달성 가능

---

## 데이터 소스

| API | 용도 | 비용 |
|-----|------|------|
| [Ambee Pollen API](https://www.getambee.com/) | 꽃가루 지수 (글로벌) | 무료 tier 있음 (100 req/day) |
| [OpenWeatherMap](https://openweathermap.org/) | 날씨 보조 데이터 | 무료 tier 충분 |
| [Google Maps Geocoding](https://developers.google.com/maps) | 위치→도시명 변환 | 무료 tier 충분 |

- Ambee 무료 tier: 일 100요청 → 유저 100명 이하에선 무료
- 유저 증가 시 Ambee 유료 플랜 ($29/월~) 또는 캐싱 전략으로 비용 최소화

## 기술 스택 (추천)

| 영역 | 선택 | 이유 |
|------|------|------|
| 앱 프레임워크 | React Native + Expo | iOS/Android 동시 개발, 빠른 출시 |
| 광고 | AdMob (react-native-google-mobile-ads) | 표준 |
| 푸시 알림 | Firebase Cloud Messaging (무료) | 위치 기반 세그먼트 가능 |
| 백엔드 | Supabase 또는 없음 (API 직접 호출) | 서버 유지 비용 최소화 |
| 배포 | Expo EAS Build | 쉬운 앱스토어 배포 |

## 디자인 방향

- 디자이너 없이 가능한 수준 유지
- 색상 코드로 위험도 표현: 🟢 낮음 / 🟡 보통 / 🟠 높음 / 🔴 매우높음
- 날씨 앱 스타일 (숫자 크게, 아이콘 단순하게)
- shadcn/ui 또는 NativeWind + Tailwind로 컴포넌트 빠르게 구성
